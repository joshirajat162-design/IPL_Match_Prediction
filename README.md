# 🏏 IPL Match Winner Prediction

## Project Overview

IPL Match Winner Prediction is a Machine Learning project that predicts the winning team of an IPL match using historical IPL data.

The model uses match-related features such as:

* Team 1
* Team 2
* Toss Winner
* Toss Decision
* Venue

and predicts the likely winner of the match.
